从零开始

安装

Hyo 现已登陆npm管理器。 使用命令行管理器进入您的项目文件夹，运行如下指令

  • yarn add hyo - 使用 yarn 安装
  • npm install --save hyo - 使用 npm 安装

安装完成后在您的html文件中导入 style.min.css

  • <link rel="stylesheet" href="node_modules/hyo/style.min.css">

您的第一个Hyo表格

只需要输入两个 props，您就可以生成属于您的第一个 Hyo 表格了。

  import Table from 'hyo';

  const def = [
    { key： "name", label： "Name", flexGrow： 1, resizable： true },
    { key： "gender", label： "Gender", flexGrow： 1 },
  ];

  const data = [
    { name： "Mars", gender： "Male" },
    { name： "Tony", gender： "Male" },
  ];

  return <Table def={def} data={data} />;`
  • 更多的例子请点击 liveDemo.


API

Table

def

类型： Array

Table 组件需要 def props 来定义每一行需要如何渲染，具体会在下文中提到。

data

类型： Array

data props 应包含所有您需要显示在 Table 组件中的内容。注意如果要改变表单中的内容，改变时必须同时将该 isLoading 设定为 true，改变完成后再设定回 false。

width

类型： Number

width props 定义了表格的宽度，溢出部分通过滚动条控制。

height

类型： Number

height props 定义了表格的高度，溢出部分通过滚动条控制。需注意过滤控制 UI 与分页控制 UI 的高度不被计算在内。

headerHeight

类型： Number

表单头部的高度。

rowHeight

类型： Number

表单内容每一行的高度。

filterable

类型： Boolean

控制过滤控制 UI 的显示。

pagination

类型： Boolean

控制分页控制 UI 的显示。

pageSize

类型： Number

如果 pagination 被设定为 true, pageSize 将定义每一页显示的行数。

isLoading

类型： Boolean

控制载入动画的显示。注意如果要改变表单中的内容，改变时必须同时将该 prop 设定为 true，改变完成后再设定回 false。

def

  • def prop 定义了每一列应当如何被渲染。

key

类型： String

该列应当显示的数据，对应 data prop中的每一列名称。例如：'name'

label

类型： String

在表单头部显示的列名称。

width

类型： Number

该列的宽度，默认为 125px 。

flexGrow

类型： Number

指明该列的弹性拉伸因子。该参数控制表格多余宽度中，每一列所占宽度的比例。

resizable

类型： Boolean

定义该列是否可以通过表单头部控制宽度

renderer

类型： Function 参数： value, row

该函数需返回一个组件，定义该列每一格应当如何显示。该函数提供两个参数。value参数是该格的内容，对应data props 中该行的数据格式。 row参数提供该行其他列的数据。

sortable

类型： Boolean

定义是否能排序该列来控制表单内容显示顺序。

onSort

类型： Function 参数： rows, key, direction

定义该列的排序方法。见例：

  const onSort = (rows, key, direction) => {
    return rows.sort((a, b) => {
      const attr1 = a[key];
      const attr2 = b[key];
      const order = !attr1 ? -1 ： !attr2 ? 1 ： attr1.toString().localeCompare(attr2);
      return direction === 'desc' ? order ： -order;
    });
  };

filterable

类型： Boolean

定义是否能通过该列来过滤表单。

filterType

类型： String or Function    参数： filter, updateFilter, removeFilter

你可以设定过滤控制 UI 为 'input' （直接输入）或 'select' （下拉选择）。此外您还可以选择设定为一个返回过滤 UI 组件的函数。该函数提供三项参数，这里是一个filterType 函数的例子：

  const filterType = (filter, updateFilter, removeFilter) => {
    return defaultFilter = (
      <div className="text-filter" key={`${filter.key}-filter`}>
        <input
          type="text"
          id={`${filter.key}-filter`}
          onChange={e => updateFilter(filter.key, e.target.value)}
          placeholder={`${filter.label}`}
        />
        <span className="clear-btn" onClick={() => removeFilter(filter.key)} />
      </div>
    );
  };

onFilter

类型： Function 参数： rows, keyword, key

onFilter 定义该列如何过滤表单。见例：

  const ageOnFilter = (rows, keyword, key) => {
    return rows.filter((row) => {
      return row[key].toString().toLowerCase().includes(keyword.toLowerCase());
    });
  };

editable

类型： Boolean

定义该列是否能在行内编辑。

editType

类型： String

定义行内编辑的类型，可设置为 'input' （直接输入）或 'select' （下拉选择）。

editOptions

类型： Array

如果editType 被设定为 'select' , 您可以制定下来选择的选项，例如：

  {
    ...
    editOptions： [
      { key： "Male", label： "Male" },
      { key： "Female", label： "Female" },
    ],
    ...
  }

onEdit

类型： Function 参数： row, col, rowId, value

此函数将在行内编辑完成时触发，见例：

  const updateValue = (row, col, rowId, value) => {
    const newData = [...this.state.data];
    const rowToUpdate = newData.find(d => d.id === row.id);
    if (rowToUpdate[col].toString() !== value.toString()) {
      rowToUpdate[col] = value;
      this.setState({ isLoading： true });
      setTimeout(() => this.setState({ data： newData, isLoading： false }), 300);
    }
  };
